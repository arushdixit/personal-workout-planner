# Muscle Group Images - Download List

## Required Images (17 total)

You need to download or create the following muscle group icon images:

1. **abs.png** - Abdominal muscles (six-pack area)
2. **adductors.png** - Inner thigh muscles
3. **biceps.png** - Front upper arm muscles
4. **calves.png** - Lower leg back muscles
5. **chest.png** - Pectoral muscles
6. **deltoids.png** - Shoulder muscles
7. **forearm.png** - Lower arm muscles
8. **glutes.png** - Buttocks muscles
9. **hamstrings.png** - Back of thigh muscles
10. **lats.png** - Latissimus dorsi (wide back muscles)
11. **lower-back.png** - Lower back/lumbar muscles
12. **neck.png** - Neck muscles
13. **obliques.png** - Side abdominal muscles
14. **quadriceps.png** - Front thigh muscles
15. **traps.png** - Trapezius (upper back/neck area)
16. **triceps.png** - Back upper arm muscles
17. **tibialis.png** - Shin muscles

## Image Specifications

- **Format**: PNG with transparent background
- **Size**: 256x256px (displayed at 48x48px, so high DPI)
- **Style**: Clean, modern anatomical illustrations
- **Color**: Single color (white or light gray) for easy theming with CSS filters
- **Background**: Transparent

## Where to Download

### Option 1: Flaticon (Recommended)
1. Visit https://flaticon.com/
2. Search for "muscle anatomy" or specific muscle names
3. Download in PNG format, 256px size
4. Choose icons with transparent backgrounds
5. Ensure consistent style across all icons

### Option 2: The Noun Project
1. Visit https://thenounproject.com/
2. Search for each muscle name
3. Download PNG format
4. Free account required for downloads

### Option 3: AI Generation
Use an AI image generator like DALL-E or Midjourney with this prompt:

```
minimalist anatomical icon of [MUSCLE_NAME], white silhouette on transparent background, medical illustration style, simple clean design, 256x256px
```

Replace [MUSCLE_NAME] with each muscle from the list above.

### Option 4: Manual Design
Use Figma or Illustrator to create a consistent set:
1. Create 256x256px artboard
2. Draw simplified anatomical shapes
3. Use single color (white)
4. Export as PNG with transparency

## Installation

Once downloaded, place all 17 PNG files in:
```
/public/images/muscles/
```

The app will automatically use these images. If an image is missing, it will fall back to emoji icons.

## Testing

After adding the images:
1. Open the app
2. Navigate to any exercise detail page
3. Click the "Target" tab
4. You should see the muscle icons displayed with your images
